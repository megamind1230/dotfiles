# pdfdash — Implementation Notes

## Overview

Interactive terminal dashboard for common PDF operations using `fzf` + `pdftk` + `poppler-utils` + `ghostscript`.

## Features (implemented)

- [X] Pick PDF with fzf preview (file size, page count, text preview)
- [X] Extract images (PNG/JPEG/TIFF) — all pages or custom range
- [X] Extract PDF pages (subset of pages → new PDF)
- [X] Merge multiple PDFs (multi-select via fzf -m)
- [X] Compress PDF (screen/ebook/printer/prepress via ghostscript)
- [X] Encrypt PDF (owner + optional user password)
- [X] Decrypt PDF (remove password protection)
- [X] Rotate pages (90° CW / 90° CCW / 180°)

## Removed (pandoc-dependent)

- [ ] PDF → Markdown  — requires pandoc (broken Haskell lib chain)
- [ ] PDF → Org       — requires pandoc (broken Haskell lib chain)

## Future Ideas

- [ ] OCR mode (ocrmypdf)
- [ ] Split PDF by page ranges into separate files
- [ ] Batch mode — operate on all PDFs in a directory
- [ ] Undo last operation / backup originals
- [ ] Gum-based UI for colored panels (if gum is available)
- [ ] Progress bar for long operations (gs compress, batch jobs)
- [ ] Config file for default settings (compression preset, image format, etc.)

## Development History

1. Basic dashboard — extract pages, extract images (PNG only), merge, pandoc conversions
2. Added fzf preview panel (file size, page count, text preview)
3. Added image format selection (PNG/JPEG/TIFF)
4. Added Compress, Encrypt, Decrypt, Rotate
5. mktemp for unique temp files
6. Auto-.pdf suffix on merge output
7. Removed pandoc-dependent features (Haskell lib chain breakage)
8. #baka comment markers for ambiguous code

## Technical Debt

- `set -euo pipefail` aborts on any failure
- Ctrl+D at password prompt exits the script
- No cleanup trap for TMPDIR
- mktemp files orphaned if script crashes
